# NumPy RL Racer

A from-scratch Deep Reinforcement Learning project where a small autonomous car learns to drive on a 2D race track.

The goal is to implement the environment, neural networks, and RL algorithms using NumPy only.

## Project constraints

Runtime dependencies:
- numpy
- matplotlib

Forbidden ML/RL dependencies:
- torch
- tensorflow
- jax
- gymnasium
- stable-baselines3

## Planned roadmap

1. Project skeleton
2. 2D car physics
3. Track representation
4. Racing environment API
5. Visualization
6. NumPy neural network
7. DQN from scratch
8. Training and evaluation scripts
